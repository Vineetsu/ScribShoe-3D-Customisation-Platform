## 🧩 What is ScribShoe?

**ScribShoe** is a stylish and interactive 3D shoe configurator built with **React**, **Three.js**, and **Tailwind CSS**. It allows users to customize shoe colors, decals, and textures in real time—and is designed for eCommerce integration with a checkout experience coming soon.

---

## ✨ Features

| Feature                  | Description                                                               |
|--------------------------|---------------------------------------------------------------------------|
| 🎨 Real-time 3D Config   | Customize shoe color, texture, and decals live                            |
| 🌀 Scroll-based Layout   | Immersive product storytelling with scroll animation                       |
| 💎Well structured UI/UX         | Clean layout, soft shadows, smooth animations                             |
| 🧵 Tailwind + ShadCN UI  | Consistent design with utility-first styling and component library         |
| 🧠 State Management      | Centralized config state using React hooks and context                     |

## 🛠️ Tech Stack

- **Frontend:** React, Vite, Tailwind CSS
- **3D Engine:** Three.js, @react-three/fiber, @react-three/drei
- **State:** React Hooks, Zustand or Context API
- **UI Components:** ShadCN/UI
